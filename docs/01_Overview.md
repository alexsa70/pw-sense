# Обзор проекта KalSense E2E Automation

## 📋 Основная информация

**Название:** kalsense-automation  
**Версия:** 1.0.0  
**Автор:** AlexL  
**Фреймворк:** Playwright 1.58.0  
**Язык:** TypeScript

## 🎯 Цель проекта

Автоматизация end-to-end тестирования веб-приложения KalSense, включая:

- ✅ Аутентификацию пользователей
- ✅ Управление медиафайлами (галерея, альбомы)
- ✅ Загрузку и обработку изображений
- ✅ AI-ассистента (Assist)
- ✅ Работу с тегами и метаданными

## 🏗️ Структура проекта

```
pw-sense/
├── .github/workflows/      # CI/CD конфигурация
├── config/                 # Конфигурация окружения
├── fixtures/               # Тестовые данные (JSON)
├── pages/                  # Page Object Model классы
├── tests/                  # E2E тестовые сценарии
├── test-files/            # Файлы для тестирования загрузки
├── utils/                 # Утилиты и хелперы
├── playwright.config.ts   # Конфигурация Playwright
├── tsconfig.json          # Конфигурация TypeScript
└── package.json           # Зависимости и скрипты
```

## 🎨 Паттерн проектирования

**Page Object Model (POM)** - разделение логики тестов и взаимодействия с UI

**Преимущества:**
- Разделение concerns
- Переиспользование кода
- Легкость поддержки
- Читаемость тестов

**Иерархия классов:**
```
BasePage (базовые методы)
  ├── LoginPage (логин)
  ├── MediaPage (медиа, загрузка)
  ├── AlbumPage (альбомы)
  ├── ImageDetailsPage (детали изображения)
  └── AssistPage (AI-ассистент)
```

## 🚀 Быстрый старт

### Установка

```bash
# Клонирование репозитория
git clone <REPOSITORY_URL>
cd pw-sense

# Установка зависимостей
npm install

# Установка браузеров Playwright
npx playwright install
```

### Конфигурация

```bash
# Создать .env файл
cp .env.example .env

# Заполнить переменные
BASE_URL=https://kal-sense.prod.kaleidoo-dev.com
USERNAME=ваш_логин
PASSWORD=ваш_пароль
```

### Запуск тестов

```bash
# Все тесты
npm test

# С видимым браузером
npm run test:headed

# Конкретный тест
npm run test:login
npm run test:assist
npm run test:album
npm run test:upload
```
